# Mini POS System (CASA)

POS (Point of Sale) - це касова програма для роботи з товарами та продажами.

## Поточний стан

Проєкт спеціально залишений як незавершена заготовка. Готова тільки частина Яріка: штрихкоди, JSON-файли, папки і README.

## Зроблено

- генерація унікального EAN-13 штрихкоду;
- створення файлу штрихкоду у папці `barcodes/`;
- створення папок `data/` і `barcodes/`;
- збереження товарів у `data/products.json`;
- завантаження товарів з `data/products.json`;
- збереження продажів у `data/sales.json`;
- завантаження продажів з `data/sales.json`;
- базове Tkinter-вікно з вкладками-заглушками.

## Не зроблено

- повна вкладка `Товари`;
- додавання товару через інтерфейс;
- перевірка назви, ціни і кількості;
- пошук товару по штрихкоду;
- повна вкладка `Каса`;
- додавання товару у чек;
- видалення товару з чеку;
- підрахунок суми чеку;
- знижка;
- завершення продажу;
- зменшення залишку товару після продажу;
- повна вкладка `Історія продажів`;
- таблиці `ttk.Treeview`;
- нормальне оновлення інтерфейсу після дій касира.

## Встановлення

Tkinter входить у стандартний Python.

Для генерації справжніх PNG штрихкодів встановіть бібліотеки:

```bash
pip install python-barcode pillow
```

Якщо бібліотеки не встановлені, програма все одно працює, але замість картинки створює файл-заглушку.

## Запуск

```bash
python main.py
```
