# Mini POS System (CASA)

POS (Point of Sale) - це касова програма для роботи з товарами та продажами.

## Можливості

Програма вміє:

- додавати товари;
- генерувати EAN-13 штрихкод для товару;
- створювати PNG картинку штрихкоду;
- шукати товар по штрихкоду;
- додавати товар у чек;
- видаляти товар із чеку;
- рахувати суму;
- робити знижку у відсотках;
- завершувати продаж;
- зберігати товари у `data/products.json`;
- зберігати історію продажів у `data/sales.json`.

## Вкладки

### Товари

Адміністратор додає товар:

- назва товару;
- ціна;
- кількість;
- штрихкод.

Кнопки:

- `Generate barcode`
- `Add product`
- `Save products`
- `Load products`

Після натискання `Generate barcode` програма створює унікальний код, наприклад `4820001234567`, і генерує файл штрихкоду в папці `barcodes/`.

### Каса

Касир вводить або сканує штрихкод товару.

Кнопки:

- `Add to receipt`
- `Remove item`
- `Clear receipt`
- `Finish sale`

Таблиця чеку:

```text
Назва | Ціна | Кількість | Сума
```

Знизу показується загальна сума:

```text
Total: 1450 грн
```

### Історія продажів

У вкладці видно:

- дату;
- суму продажу;
- кількість товарів.

## Встановлення

Tkinter входить у стандартний Python.

Для генерації справжніх PNG штрихкодів встановіть бібліотеки:

```bash
pip install python-barcode pillow
```

Якщо бібліотеки не встановлені, програма все одно працює, але замість картинки створює файл-заглушку.

## Запуск

```bash
python main.py
```

## Як користуватись

1. У вкладці `Товари` введіть назву, ціну та кількість.
2. Натисніть `Generate barcode`, щоб отримати унікальний EAN-13 код.
3. Натисніть `Add product`, щоб додати товар і зберегти його.
4. У вкладці `Каса` введіть або відскануйте штрихкод.
5. Натисніть `Add to receipt`.
6. За потреби введіть знижку.
7. Натисніть `Finish sale`, щоб завершити продаж.
